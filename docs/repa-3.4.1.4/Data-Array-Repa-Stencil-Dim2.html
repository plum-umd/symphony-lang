stencil2 :: QuasiQuoter Source #

QuasiQuoter for producing a static stencil defintion.

A definition like

    [stencil2|  0 1 0
                1 0 1
                0 1 0 |]

Is converted to:

    makeStencil2 (Z:.3:.3)
       (\ix -> case ix of
                 Z :. -1 :.  0  -> Just 1
                 Z :.  0 :. -1  -> Just 1
                 Z :.  0 :.  1  -> Just 1
                 Z :.  1 :.  0  -> Just 1
                 _              -> Nothing)
